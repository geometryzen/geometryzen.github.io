Frequently Asked Questions

Search


  1. Why can't I see the WebGL 3D visualizations?

    You probably aren't using a web browser that supports WebGL (Web Graphics Library).

    To check if your browser supports WebGL, go to the "Your browser supports WebGL" page. This resource will also guide you to the support site for your browser.

    Windows PC Users: WebGL does not run in older versions of Microsoft Internet Explorer. Microsoft has confirmed that it will be supporting WebGL for IE11 in Windows 8.1. For now, you should use Google Chrome or Mozilla Firefox, or Opera Next as your web browser.

    Mac PC Users: Safari does support WebGL, but it must be configured to do so. How do I enable WebGL in Safari?

    Mobile Users: WebGL may not be supported on your mobile device.

  2. Can I import a Python module that is not baked into the Geometry Zen software?

    There are plans to be able to include a module by specifying the resource URL.

  3. How do I see documentation for Python modules?

    The process for module documentation is being considered. For now, it will be necessary to follow examples or view the source code.

What is Geometry Zen in a nutshell?

Geometry Zen is a free, online, open-source, and collaborative tool for Computational Modeling combining the Python programming language, WebGL 3D visualization and Geometric Algebra.
